NatCorder.CommitSamples

public static void CommitSamples (float[] sampleBuffer, long timestamp);

Description

Commit an audio sample buffer for encoding.

The sample buffer is expected to contain raw PCM sample data, interleaved by channel. Each sample is expected to be a float in the range [-1.0f, 1.0f].

This function is expected to be invoked by implementations of IAudioSource. The audio source provides NatCorder information about the audio sample rate and channel count. 

private void OnAudioFilterRead (float[] data, int channels) { // Invoked by Unity/MonoBehaviour
    // Send to NatCorder for encoding
    NatCorder.CommitSamples(data, (long)(AudioSettings.dspTime * 1e+9f));
}
